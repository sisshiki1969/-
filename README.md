# 外来・在宅ベースアップ評価料 シミュレータ

医科診療所（無床／有床）向けに、外来・在宅ベースアップ評価料 (Ⅰ)(Ⅱ) の収支試算と区分判定を行う、完全クライアントサイドの静的ウェブツールです。

## 特長

- 評価料(Ⅰ) の初診・再診・訪問診療料への加算額（年額）を自動算定
- 評価料(Ⅱ) の該当区分（イ〜チ：1〜8点）を入力値から自動判定
- 必要賃金改善額に対する過不足・収入差額を可視化
- 入力データはブラウザ内で完結（外部送信なし）
- 印刷／PDF 出力に最適化したスタイル

## 算定根拠

- 評価料(Ⅰ)：初診 6点／再診 2点／訪問診療料Ⅰ・1（同一建物以外）28点／訪問診療料Ⅰ・2 等（同一建物）7点
- 評価料(Ⅱ) 区分判定式：`8 × (賃金改善必要額 − (Ⅰ)算定額) ÷ (10 × 月間外来在宅延患者数 × 8)` の小数点以下を切り上げた値が必要区分点数

## 使い方

1. リポジトリをクローンして `index.html` をブラウザで開くだけで動作します。
2. GitHub Pages 等でそのまま静的ホスティング可能です。

```sh
# ローカル確認（任意：簡易サーバ）
python3 -m http.server 8000
# → http://localhost:8000
```

## 技術スタック

- HTML + Tailwind CSS (CDN)
- Vanilla JavaScript（ビルド不要）

## 免責

本ツールの試算結果は厚生労働省告示・通知の公開情報に基づく目安です。実際の届出・算定にあたっては、必ず最新の通知および地方厚生(支)局の指導内容をご確認ください。
